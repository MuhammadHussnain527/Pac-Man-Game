# 🎮 Pac-Man Game (Java)

This is a simple **console-based Pac-Man game** developed in Java using Object-Oriented Programming concepts.

---

## 🚀 Features
- Player-controlled Pac-Man  
- Random moving ghost  
- Score system  
- Lives system  
- Collision detection  
- Input validation  

---

## 🎮 Controls
- W → Up  
- S → Down  
- A → Left  
- D → Right  
- Q → Quit  

---

## 🧱 Symbols
- `#` → Wall  
- `.` → Food  
- `P` → Player  
- `G` → Ghost  

---

## 🧠 Concepts Used
- Classes and Objects  
- Encapsulation  
- Arrays (2D Maze)  
- Loops and Conditions  
- Random class  
- Scanner (input)  

---

## ⚙ How to Run

### Step 1: Compile

java Main.java


### Step 2: Run

java Main
